# Boutique Management System

## Description
A boutique management system developed using C++ to manage clothing inventory and shopping cart operations.

## Features
- Add multiple clothing items
- Manage item categories
- Track shirt types, colors, and sizes
- Monitor item quantities
- Display inventory summaries

## Technologies Used
- C++
- STL Vector
- Structures (Struct)
- Object-Oriented Programming Concepts
